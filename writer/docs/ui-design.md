## 1. 整体布局

页面采用四栏式布局，从左到右依次为：

```
┌──────────┬──────────────┬──────────────────────────────────┬──────────────┐
│          │              │            菜单栏                │              │
│          │              ├──────────────────────────────────┤              │
│          │              │                                  │              │
│  类目    │   详情面板   │        小说内容区                │   当前状态   │
│  管理器  │  (选中时     │    (章节名 + 段落列表)           │    面板      │
│  (树形)  │   显示)      │                                  │              │
│          │              ├──────────────────────────────────┤              │
│          │              │        聊天框区域                │              │
│          │              │  (状态栏 + 输入框 + 按钮)        │              │
└──────────┴──────────────┴──────────────────────────────────┴──────────────┘
```

### 1.1 尺寸设计

| 区域 | 宽度 | 说明 |
|------|------|------|
| 类目管理器 | 250px (可拖拽调整) | 最小 200px，最大 400px |
| 详情面板 | 300px (选中时显示) | 未选中时隐藏 |
| 小说编写区 | flex: 1 (自适应) | 最小宽度 400px |
| 当前状态面板 | 280px (可拖拽调整) | 最小 200px，最大 400px |

### 1.2 菜单栏
- 位置：小说编写区和当前状态面板的顶部（横跨中间和右侧两栏）
- 高度：40px
- 包含菜单项：文件、AI 配置

## 2. 类目管理器（左侧面板）

### 2.1 树形结构

```
┌─────────────────────────┐
│ 📂 类目管理器      [+]  │
│─────────────────────────│
│ ▶ 人物                  │
│   ├── 主角              │
│   │   ├── 张三          │
│   │   └── 李四          │
│   └── 反派              │
│       └── 王五          │
│ ▶ 地点                  │
│   ├── 修仙界            │
│   └── 凡人界            │
│ ▶ 宗门                  │
│   ├── 天剑宗            │
│   └── 万法宗            │
│ ▶ 物品                  │
│   └── 神兵              │
└─────────────────────────┘
```

### 2.2 交互说明
- 点击 `▶` / `▼` 展开/折叠子节点
- 点击节点文本选中该节点
- 选中节点高亮显示
- 每个节点右侧显示操作按钮（hover时显示）：
  - `+` 添加子节点
  - `✏️` 编辑
  - `🗑️` 删除
- 顶部 `[+]` 按钮添加新的顶级大类
- 支持右键菜单（添加子项、编辑、删除）

### 2.3 详情面板（选中项的右侧列）

```
┌──────────────────────────────┐
│ 人物 > 主角 > 张三       [×] │

│ │年龄: 18              │     │
│ │境界: 筑基期          │     │
│ │性格: 坚韧不拔        │     │

│                              │
```

- 顶部面包屑导航显示当前项的路径
- `[×]` 关闭详情面板
- 属性为自由文本（键值对格式），用户可自行定义

### 5.1.1 段落交互
- 点击段落选中，显示高亮边框
- 选中段落显示绑定信息标签
- 段落可编辑（双击或按 Enter 进入编辑模式）
- 选中段落时，标签区显示绑定的类目信息

## 7. 配色方案

| --warning | #f59e0b | 警告/进行中 |

## 8. 响应式设计

- 最小支持宽度: 1200px
- 低于 1200px 时，类目管理器和当前状态面板可通过按钮切换显示/隐藏
- 弹窗（模态对话框）始终居中显示，宽度自适应（最大 800px）
